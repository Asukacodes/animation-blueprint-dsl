## How to capture the preview image:

1. Start the editor:
   ```bash
   cd editor
   python -m http.server 8080
   ```

2. Open http://localhost:8080 in Chrome/Edge

3. Load the example: Click "加载示例" to see a sample state machine

4. Take a screenshot and save as `preview.png` (recommended size: 1280x720)

5. Save to: `docs/preview.png`

## Screenshot guidelines:
- Show the editor with the example ABP_Player state machine loaded
- Include the toolbar at top
- Show at least 2-3 states with a transition
- Use Chrome with dark theme for best appearance
