# GitHub Pages Configuration

This repository is configured for GitHub Pages hosting.

**Live Demo:** https://Asukacodes.github.io/animation-blueprint-dsl/

## Setup Instructions

### 1. First Push (manual)
```bash
cd animation-dsl
git push -u origin master
```

### 2. Enable GitHub Pages
1. Go to: https://github.com/Asukacodes/animation-blueprint-dsl/settings/pages
2. Source: Deploy from a branch
3. Branch: master, / (root)
4. Click Save

### 3. Add Screenshot
Take a screenshot of the editor (http://localhost:8080) and save as `docs/preview.png`

### 4. Access
After ~2 minutes, your site will be live at the URL above.
